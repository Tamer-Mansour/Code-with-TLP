# 32-bit Integer to IP Address

Read a single non-negative integer from standard input. Convert it to its dotted-decimal IPv4 address representation and print the result.

The integer is in the range 0 to 4294967295 (0x00000000 to 0xFFFFFFFF).

## Examples

Input: `3232235777`
Output: `192.168.1.1`

Input: `0`
Output: `0.0.0.0`

Input: `4294967295`
Output: `255.255.255.255`

Input: `134744072`
Output: `8.8.8.8`

Input: `2130706433`
Output: `127.0.0.1`
