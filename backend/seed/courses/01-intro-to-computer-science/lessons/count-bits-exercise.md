# Exercise: Count the 1 Bits

Practice your understanding of binary representation by writing a program that counts how many `1` bits appear in the binary form of each input number.

Given a non-negative integer, its **bit count** (also called *Hamming weight* or *popcount*) is the number of positions where the binary digit is `1`.

Examples:
- `7` in binary is `111` → 3 ones
- `10` in binary is `1010` → 2 ones
- `255` in binary is `11111111` → 8 ones
- `0` in binary is `0` → 0 ones

Read one integer per line until EOF and print the bit count for each.

## Sample Input

```
7
255
0
10
```

## Sample Output

```
3
8
0
2
```
