# Exercise: Stack Command Processor

Implement a `Stack` class that supports push, pop, peek, and size operations, then process a sequence of commands from standard input.

This exercise reinforces:

- Implementing a data structure as a class with private state.
- Handling edge cases (empty stack) gracefully with correct output.
- Writing clean, encapsulated code where the internal list is never exposed directly.

See the problem statement for the exact command format and expected output.
