# Exercise: Shape Area Calculator

In this exercise you will build a small shape hierarchy using inheritance and method overriding.

Define a base class `Shape` with an `area()` method, then create subclasses `Circle`, `Rectangle`, and `Triangle` that each override `area()` with the correct formula.

A command loop reads shapes and AREA queries from standard input. This exercise reinforces:

- Defining a class hierarchy with a common interface.
- Overriding methods in subclasses.
- Using polymorphism to call the right implementation at runtime.

See the problem statement for the exact input/output format.
