# Exercise: Bank Account Command Processor

In this exercise you will implement a `BankAccount` class and a command-processing loop.

Your class must support three operations — deposit, withdraw, and balance inquiry — responding correctly to each command read from standard input.

This exercise tests your ability to:

- Define a class with instance state (`balance`).
- Implement methods that read and modify that state.
- Handle invalid operations gracefully (insufficient funds).

See the problem statement for the exact input/output format.
