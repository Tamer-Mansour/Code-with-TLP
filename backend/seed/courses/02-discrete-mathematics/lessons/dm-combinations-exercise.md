# Exercise: Combinations Calculator

Given `n` and `r`, compute `C(n, r) = n! / (r! * (n-r)!)` — the number of ways to choose `r` items from `n` without regard to order.

Your program must read two integers from standard input and output the exact integer value of `C(n, r)`.

Use only the Python standard library. No floating-point operations — compute the exact integer result.
