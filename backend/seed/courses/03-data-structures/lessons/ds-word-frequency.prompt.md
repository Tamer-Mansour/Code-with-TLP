# Word Frequency Counter

Given a list of words, count how many times each word appears. Print the words and their counts in **alphabetical order**.

## Input Format

- Line 1: a single integer `n` — the number of words.
- Line 2: `n` words separated by spaces (all lowercase, letters only).

## Output Format

For each distinct word (in alphabetical order) print: `word count` on its own line.

## Constraints

- `1 <= n <= 10000`
- Each word is 1–20 lowercase letters.

## Examples

**Example 1**
```
Input:
7
apple banana apple cherry banana apple cherry

Output:
apple 3
banana 2
cherry 2
```

**Example 2**
```
Input:
3
dog cat dog

Output:
cat 1
dog 2
```
