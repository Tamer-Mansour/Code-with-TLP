# BST Insert and In-Order Traversal

Build a Binary Search Tree by inserting integers one at a time, then print the values produced by an in-order traversal (which should be in sorted order).

## Input Format

- Line 1: a single integer `n` — the number of values to insert.
- Line 2: `n` integers separated by spaces (no duplicates).

## Output Format

Print the in-order traversal as space-separated integers on a single line.

## Constraints

- `1 <= n <= 500`
- Values are distinct integers in the range [-10000, 10000].

## Examples

**Example 1**
```
Input:
5
5 3 7 1 4

Output:
1 3 4 5 7
```

**Example 2**
```
Input:
3
10 5 15

Output:
5 10 15
```

**Example 3**
```
Input:
1
42

Output:
42
```
