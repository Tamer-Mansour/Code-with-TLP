# Reverse a Linked List

Given the values of a singly linked list, reverse the list and print the node values from head to tail.

## Input Format

- Line 1: a single integer `n` — the number of nodes.
- Line 2: `n` integers separated by spaces — the node values from head to tail.

## Output Format

Print the reversed list as space-separated integers on a single line.

## Constraints

- `1 <= n <= 1000`
- Node values are integers in the range [-10000, 10000].

## Examples

**Example 1**
```
Input:
5
1 2 3 4 5

Output:
5 4 3 2 1
```

**Example 2**
```
Input:
1
42

Output:
42
```

**Example 3**
```
Input:
4
10 20 30 40

Output:
40 30 20 10
```
