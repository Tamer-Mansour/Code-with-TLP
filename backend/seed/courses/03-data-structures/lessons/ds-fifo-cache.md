# Exercise: FIFO Cache Simulator

Simulate a fixed-capacity FIFO (First-In-First-Out) cache. When the cache is full and a new key is accessed, evict the key that has been in the cache the longest.

This exercise reinforces queue-based eviction and hash-table-backed O(1) lookup — the building blocks of real caching systems.
