# Exercise: Two-Sum with Hash Table

Given an array of integers and a target sum, find all unique pairs of indices (i, j) with i < j such that arr[i] + arr[j] == target.

This exercise demonstrates the classic hash-table pattern of trading O(n) space for O(n) time — reducing a brute-force O(n²) problem to a single pass.
