# Exercise: Bitwise Operations Evaluator

Practice the bitwise operations used in logic gates and ALU design by implementing a simple expression evaluator.

## Background

The five basic bitwise operations on integer bit patterns mirror the logic gates covered in Module 1:

- **AND**: both bits must be 1.
- **OR**: at least one bit must be 1.
- **XOR**: bits must differ.
- **NAND**: NOT(AND) — result masked to W bits.
- **NOR**: NOT(OR) — result masked to W bits.

These are the exact operations performed by ALU gates on integer registers.

## Your Task

Read two integers A, B and an operation name from stdin. Print the result.
