# Exercise: Two's Complement Converter

Practice the two's complement negation algorithm by implementing it programmatically.

## Background

To negate a number in two's complement:
1. Write N in binary with exactly W bits.
2. Flip every bit (bitwise NOT).
3. Add 1 to the result.
4. Take the lower W bits.

This gives the W-bit two's complement representation of −N.

## Your Task

Read two integers N and W from stdin. Output the W-bit two's complement binary string for −N.
